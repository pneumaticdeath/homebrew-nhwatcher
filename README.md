# homebrew-nhwatcher

Homebrew tap for [NH Watcher](https://github.com/pneumaticdeath/NHWatcher) — a macOS screensaver that displays live NetHack games from nethack.alt.org.

## Installation

```bash
brew tap pneumaticdeath/nhwatcher
brew install --cask nhwatcher
```

## Updating

```bash
brew upgrade --cask nhwatcher
```

## Uninstalling

```bash
brew uninstall --cask nhwatcher
brew untap pneumaticdeath/nhwatcher
```
